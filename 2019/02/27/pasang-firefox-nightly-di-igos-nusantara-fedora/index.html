Firefox Nightly sudah pada tahukan? ok gak usah basah-basahan lagi karena kita semua sudah tahu kalau Nightly itu versi untuk para Q&A dan Developer. Well, postingan ini sederhana karena lagi sendiri tidak ada yang dikerjakan maka terciptalah niat buat nyampah di blog ini, tulisannya gak akan jauh beda dari title diatas “Pasang Firefox Nightly di IGOS Nusantara / Fedora“.

Alasan saya menulis ini karena Nightly (Firefox) memiliki user interface baru yang disebut “Australis”, rupanya hampir sama dengan peramban besutan “Google” (OH NO…!!). Ok sekarang saatnya WTF ?? (Where To Find) not (What The F*CK).

Before : Firefox kita masih versi biasa seperti ini





Firefox Standard (Final)


Dan sekarang kita akan menuju perubahan karena negara api sedang menyerang, YAA the Nightly is come. Caranya adalah sebagai berikut bero…

Buka Console (masuk root ya bos..)

Buka link http://nightly.mozilla.org untuk mengunduh versi terakhir dari si malam (Nightly)

Di console gunakan command berikut ini,

# cd /opt/
# wget http://ftp.mozilla.org/pub/mozilla.org/firefox/nightly/latest-trunk/firefox-31.0a1.en-US.linux-i686.tar.bz2
# tar -xvf firefox-31.0a1.en-US.linux-i686.tar.bz2
# mv firefox nightly
# ln -s /opt/nightly/firefox /usr/local/bin/nightly

Menjalankannya? Gunakan saja command “nightly” di console bos.
