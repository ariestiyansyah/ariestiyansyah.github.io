Webmaker with SMK ITACO

ariestiyansyah
2019-02-27
Mozilla
Bekasi, Event, SMK ITACO, Webmaker, mozilla

August 18, 2014 we will carry out the webmaker event we’ve scheduled previously, the event held at SMK ITACO Bekasi, this is a vocational school for children who are less economic conditions.

We only use my laptop without a projector because the existing infrastructure in SMK ITACO still inadequate, so we learned together using my laptop. Webmaker event this time is not too long because we only schedule for a few hours and I introduced the application of what is in this webmaker and all about Mozilla and what we can do with Webmaker application.

The event starts at 1PM, we started introducing ourselves first and discuss about what is web and Open Web and why Mozilla support the open web. The first material that I convey to all students of SMK ITACO is a thimble, the enthusiasm of the participants was very high when he saw a demonstration WYSIWYG, then I invite participants to join together to try gathering and use thimble, we also do video editing using popcorn and edit websites of iswawirausaha.net using X-Ray Googles.


The main purpose of this event is to introduce Mozilla and webmaker on SMK ITACO. we are very happy to enjoy this event because many of them ask about webmaker, this is what we expect the presence of 2-way communication. want to know what we have create on 90 minutes running this webmaker? here is the list.

FYI : We are using my account to practice the webmaker because of limited resource (laptop)
